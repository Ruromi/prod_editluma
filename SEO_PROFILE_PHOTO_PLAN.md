# SEO Profile Photo Plan

## Positioning

- Product category: `Professional profile photo AI`
- Core outcome: turn a casual photo into a LinkedIn-ready, business-grade profile image
- Primary audiences: job seekers, founders, creators, freelancers, and teams with public-facing profile photos
- Message priority: result first, before / after second, workflow and features last

## Copy Guardrails

- Lead with outcomes such as `LinkedIn-ready`, `business-ready`, and `CV-ready`
- Keep `AI image generator` positioned as a supporting workflow, not the main product story
- Prefer `profile photo`, `headshot`, and `public-facing profile` over broad `image editing` language
- Do not publish unverified social proof or numeric claims

## Home Message Stack

- H1 direction: `AI Profile Photo Generator | LinkedIn Headshot AI`
- Hero promise: `Get a professional profile photo in 30 seconds`
- Supporting copy: `Turn your casual photo into a LinkedIn-ready, business-grade profile image with AI.`
- Primary CTA: `Upgrade my profile photo`
- Proof chips: `LinkedIn-ready`, `Business-ready`, `CV-ready`

## Page Direction

- `/`
  Home page should sell the result, not the tool category.
- `/ai-portrait-enhancer`
  This is the core profile-photo SEO landing and should own the strongest `LinkedIn headshot AI` and `professional headshot AI` messaging.
- `/ai-photo-enhancer`
  This page should answer the `can I rescue my current photo?` intent.
- `/ai-image-generator`
  This page should capture broader generator traffic but frame generation as the first-draft path for profile photos, not generic creative ideation.
- `/pricing`
  Pricing should stay anchored on profile-photo outcomes and public-facing profile use cases.
